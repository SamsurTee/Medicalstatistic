Interactive Medical Statistics Module Pack

This module pack is designed to engage students with interactive, hands-on exercises, an AI-assisted activity, and a quiz. Students can expand each section to explore code, instructions, and tasks.

1. Interactive SPSS/R Exercises

Exercise 1 – Descriptive Statistics & Graphs

Scenario: A researcher measures blood pressure, weight and height for 100 adults.

Variables: Age (years), Sex (Male/Female), Weight_kg, Height_cm, SystolicBP, DiastolicBP.

R Code:

set.seed(42)
n <- 100
data1 <- data.frame(
  Age = sample(20:70, n, replace=TRUE),
  Sex = sample(c("Male","Female"), n, replace=TRUE),
  Weight_kg = round(rnorm(n, 70, 12), 1),
  Height_cm = round(rnorm(n, 170, 10), 1),
  SystolicBP = round(rnorm(n, 120, 15), 1),
  DiastolicBP = round(rnorm(n, 80, 10), 1)
)
write.csv(data1, "exercise1.csv", row.names=FALSE)

SPSS Instructions: Import exercise1.csv via File → Import Data → CSV. Define variable types, then use Analyze → Descriptive Statistics to compute means and create histograms.

Tasks:

  • Compute summary statistics (mean, SD, quartiles) for each numeric variable.
  • Draw histograms for Weight_kg and SystolicBP and comment on their shapes.
  • Compare average blood pressure by sex.

2. AI-Assisted Teaching Activity

AI Activity Outline

Objective: Encourage students to use generative AI tools (e.g., ChatGPT) as collaborative assistants for coding and interpretation while maintaining critical thinking.

  1. Pre-reading discussion: Provide short articles summarising AI’s potential and limitations in education. Discuss benefits such as additional resources and perspectives, and concerns like overreliance and accuracy.
  2. Prompt-writing workshop: Students work in pairs to write prompts asking ChatGPT to produce R or SPSS syntax for one of the datasets. They then refine the prompt to improve clarity.
  3. Execution & verification: Students run the AI-generated code and compare results to manual solutions or built-in functions. They check whether the AI code follows correct statistical practice and interpret results.
  4. Critical reflection: Groups share what worked well, identify mistakes made by the AI, and describe how they corrected them.
  5. Ethics & synthetic data discussion: Discuss why synthetic datasets are useful for teaching and privacy. Mention that synthetic data help simulate research scenarios, test algorithms, and protect privacy. Also discuss limitations such as potential data leakage or bias.

Alternative perspectives: You could use other AI tools, assign AI as a “virtual student” to critique, or ask students to compare AI explanations with textbook definitions.

3. Interactive Quiz

This quiz contains 25 questions. Students can click each question to reveal the answer.

1. Define descriptive statistics vs inferential statistics.

Answer: Descriptive statistics summarise and describe features of a dataset (e.g., mean, median, standard deviation). Inferential statistics use sample data to make generalisations about a population (e.g., confidence intervals, hypothesis tests).

2. What conditions are required for using an independent samples t-test?

Answer: Two independent groups, continuous outcome variable, approximately normally distributed within each group, and equal variances (or use Welch’s test for unequal variances).

4. Implementation Notes

Dataset distribution: Provide the CSV files or allow students to run the R scripts to create them. Synthetic datasets help protect privacy and support educational use.

Software flexibility: Each exercise includes both SPSS and R instructions to build cross-platform skills.

AI activity timing: Introduce the AI-assisted activity mid-semester when students have basic R/SPSS skills.

Assessment: Use the quiz as a formative or summative assessment. Questions mix conceptual understanding and practical interpretation.

Alternative resources: Consider using publicly available synthetic datasets (e.g., Health Gym, Synthea) for more complex projects or exploring other AI tools.
